:mod:`gs.group.member.join`
===========================

Contents:

.. toctree::
   :maxdepth: 2

   HISTORY

The :mod:`gs.group.member.join` product handles joining people to
groups. It provides the *Join* page, and the *Subscribe* and
*Confirm* email-commands, as well as the basic API for joining
people to groups. This API is used by other products, such as
:mod:`gs.group.member.add.base`, :mod:`gs.group.member.request`,
:mod:`gs.profile.invite`, and :mod:`gs.profile.subscribe`.
